# react-tgrid

A React component to display data in tree structure.

## Installation

`npm install --save react-tgrid`

## Usage

```js
import TreeGrid from './js/TreeGrid';  
<TreeGrid  
  data={DATA}  
  columns={COLUMNS}
  options={OPTIONS}
  id={'id'}
  parentId={'parentId'}></TreeGrid>
```
